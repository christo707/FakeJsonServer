# FakeJsonServer

For Testing

# Getting started

Install JSON Server

npm install -g json-server

Start JSON Server

json-server --watch db.json

POST, PUT, PATCH or DELETE requests, changes will be automatically and safely saved to db.json

more info: https://github.com/typicode/json-server
